# Checklist de Implementación (Python)

- [ ] Contrato definido (HTTP o evento) con ejemplos.
- [ ] Plan de implementación por módulos/capas.
- [ ] Configuración por env/SSM; sin secretos hardcode.
- [ ] Logging con `trace_id` (si existe helper del repo).
- [ ] Tests agregados/actualizados (happy + error).
- [ ] Formato/lint según `pyproject.toml` (black/isort/mypy/ruff si aplica).
