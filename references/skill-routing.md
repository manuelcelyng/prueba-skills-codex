# Skill routing (coordinadores)

## Cuándo usar `planning` vs `planning-java`/`planning-python`
- Solo Java → `planning-java`
- Solo Python → `planning-python`
- Mezcla o no está claro → `planning` (coordina y deriva)

## Cuándo usar `dev` vs `dev-java`/`dev-python`
- Solo Java → `dev-java`
- Solo Python → `dev-python`
- Mezcla o no está claro → `dev` (coordina y deriva)
