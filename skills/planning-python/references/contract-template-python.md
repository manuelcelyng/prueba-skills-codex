# Template de contrato (Python)

    Usa este formato para `context/hu/<HU_ID>/contrato.md` o el archivo de contrato existente.

    ## Si expone HTTP (FastAPI)
    - Metodo y ruta
    - Headers
    - Request/Response con ejemplos JSON
    - Codigos de respuesta y mensajes

    ## Si es Lambda/ETL
    - Evento de entrada (payload, campos requeridos, ejemplo JSON)
    - Salidas esperadas (DB, SQS, logs) y estados logicos
    - Errores esperados con ejemplos JSON si aplica
